\section{Вариационные ряды и их характеристики}

\subsection{Генеральная и выборочная совокупности, объём выборки.}

Рассмотрим постановку задачи математической статистики: по результатам
наблюдения за некоторой случайной величиной $\xi$ требуется сделать
выводы о неизвстном законе распределения этой величины $\mathcal{L}(x,\varTheta)$
либо о неизвестных парамерах $\varTheta_1, \dots, \varTheta_n$ известного распределения.
\bigskip

Пусть $\xi$~--- случайная величина с некоторой (теоретической) функцией распределения
$F_{\xi}(x) = P\{\xi < x\}, \;\;\; x \in R.$
\bigskip

\textbf{Определение:}
\smallskip

Совокупность $n$ независимых одинаково распределённых случайных величин $\mathbf{X}_1,
\mathbf{X}_2, \dots, \mathbf{X}_n$ называется выборкой (выборочной совокупностью),
извлечённой из распределения случайной величины $\xi$.
\bigskip

\textbf{Определение:}
\smallskip

Под генеральной совокупностью понимается множество всех возможных значений случайной
величины $\xi$.
\bigskip

\textbf{Определение:}
\smallskip

Объёмом совокупности называется количество всех её элементов, объё выборки или
выборочной совокупности обозачается $n$, генеральной совокупности~--- $N$.
\bigskip

\subsection{Варианционный ряд, варианта, частота. Виды вариационных рядов.
Гистограмма, полигон.}

\textbf{Определение:}
\smallskip

Пусть $x_1, x_2, \dots, x_n$~--- выборка из генеральной совокупности значений.
\smallskip

Вариационным рядом называется последовательность $x_1^{*}, x_2^{*}, \dots, x_n^{*}$
элементов выборки расположенных в порядке неубывания, т.е. $x_1^{*} \leq x_2^{*}
\leq \dots \leq x_n^{*}.$ 
\smallskip

$x_i^{*}$~--- варианта.
\smallskip

$n_i$~--- частота появления варинты $x_i^{*}$ в выборке.
\bigskip

\textbf{Определение:}
\smallskip

Точечным вариационным рядом называется:

\begin{center}
    \includegraphics[width=150mm]{toch_var_rad_1.PNG}
\end{center}

$x_i$~--- варианта, $n_i$~--- частота соответствуующей варианты.
\smallskip

$m$~--- количество групп (различных вариант (вариант в таблице)).
\smallskip

$n = \sum^m_{i = 1} n_i$, где $n_i$~--- объём выборки.
\bigskip

Для графического представления точечных вариационных рядов используется полигон
частот~--- ломанная с вершинами в точках $(x_i, n_i)$.

    \includegraphics[width=120mm]{toch_var_rad_2.PNG}
\end{center}

\textbf{Определение:}
\smallskip

Интервальным вариационным рядом называется:

\begin{center}
    \includegraphics[width=150mm]{interval_var_rad_1.PNG}
\end{center}

$x_i$~--- варианты, $n_i$~--- частота.
\smallskip

$m$~--- количество групп (интервалов).
\smallskip

$n = \sum^m_{i = 1} n_i$, где $n_i$~--- объём выборки.
\bigskip

Для графического представления интервальных вариационных рядов используется гистограмма
частот~--- фигура, составленная из прямоугольников, одной стороной которых служат
интервалы $(x_i, x_{i + 1}]$, а длина второй равна $n_i$.

\subsection{Формулы числовых характеристик. Эмперическая функция распределения
(ЭФР). Свойства ЭФР.}

\textbf{Определение:}
\smallskip

Выборочным средним называется величина:

$$\overline{x} = \frac{1}{n} \sum^n_{i = 1} \mathbf{X_i}.$$

Если данные представлены в виде точечного или интервального вариационного ряда,
то для вычисления используют формулу:

$$\overline{x} = \frac{1}{n} \sum^n_{i = 1} x_i * n_i,$$

где $m$~--- количество групп в точечном или интервалов в интервальном вариационном
ряду, $n_i$~--- частота, т.е. количество элементов выборки, принадлежащихх $i$"=той
группе или $i$"=тому интервалу, $x_i$~--- варианта для точечного ряда и середина
$i$"=того интервала для интервального ряда.

\textbf{Определение:}
\smallskip

Выборочной дисперсией (смещённой) называется величина:

$$S^2 = \frac{1}{n} \sum^n_{i = 1} (\mathbf{X}_i - \overline{x})^2.$$

Она характеризует среднее из квадратов отклонений наблюдаемой величины от выборочного
среднего. Величина $S = \sqrt{S^2}$ называется выборочным средним квадратическим
отклонением (смещённым) величин выборки от выборочного среднего.
\smallskip

Если данные представлены в виде точечного или интервального вариационного ряда, то для
вычисления используют формулу:

$$S^2 = \frac{1}{n} \sum^m_{i = 1} (x_i - \overline{x})^2 n_i.$$

или

$$S^2 = \left(\frac{1}{n} \sum^m_{i = 1}x^2_i n _i\right) - \overline{x}^2.$$

Здесь $m$~--- количество групп в точечном или интервалов в интервальном вариационных
рядах, $n_i$~--- частота. т.е. количество элементов выборки, принадежащих
$i$"=той группе или $i$"=тому интервалу, $x_i$~--- варианта для точечного ряда и
середина $i$"=того интервала для интервального ряда.
\bigskip

\textbf{Определение:}
\smallskip

Выборочной дисперсией (несмещённой) называется величина:

$$\overline{\sigma} = \frac{1}{n - 1} \sum^n_{i = 1} (\mathbf{X}_i \overline{x})^2.$$

Аналогично, величина $\overline{\sigma} = \sqrt{\overline{\sigma}^2}$ называется
выборочным несмещённым средним квадратическим отклонением.
\smallskip

Очевидно, что смещённая и несмещённая выборочные дисперсии связаны формулой:

$$\overline{\sigma}^2 = \frac{n}{n - 1} S^2.$$

\subsection{Эмперическая функция распределения ЭФР. Свойства ЭФР.}

\textbf{Эмперическая функция распределения ЭФР}
\smallskip

Эмперической функцией распределения (ЭФР) называется функция

$$\tilde{F}_n(x) = \frac{1}{n} \sum^n_{i = 1} e (x - \mathbf{X}_i),$$

где $e(x) = 1$, при $x > 0 \;\;\; e(x) = 0$, при $x \leq 0$.
\smallskip

Таким образом, если $\mathbf{X}_i$, то $e(x) = 1$, если $\mathbf{X}_i \geq x$,
то $e(x) = 0$, а сумма $e(x - \mathbf{X}_i)$ будет равна количеству элементов выборки,
которые приняли значение, строго меньше некоторого $x \in R$.
\smallskip

Пусть $x_1, x_2, \dots, x_n$~--- реализация выборки $\mathbf{X}_1, \mathbf{X}_2,
\dots, \mathbf{X}_n$, т.е. наблюдавшиеся значения случайной величины $\xi$.
\smallskip

Обозначим $\mu(x)$~--- число элементов выборки, строго меньших $x \in R$.
Тогда эмпирическая функция распределения $\tilde{F}_n(x)$ может быть определена
как

$$\tilde{F}_n(x) = \frac{\mu(x)}{n}.$$
\bigskip

\textbf{Свойства ЭФР:}

\begin{enumerate}
    \item $0 \leq \tilde{F}_n(x) \leq 1,$ т.к. $0 \leq \mu(x) \leq n$;
    \item неубывающая непрерывная слева функция;
    \item $\tilde{F}_n(x)$~--- ступенчатая функция для всех типов распределений;
    \item $\tilde{F}_n(x)$ сходится по распределению к $F_{\xi}(x)$.
\end{enumerate}

\section{Оценки параметров распределения}

\subsection{Понятия статистики, оценки, выборочной характеристики.}

\textbf{Определение:}
\smallskip

Пусть $g(t_1, \dots, t_n)$~--- непрерывная функция. Оценкой $\varTheta$ назовём
$\tilde{\varTheta} = g (\mathbf{X}_1, \dots, \mathbf{X}_2)$.

Если $g(\mathbf{X}_1, \dots, \mathbf{X}_n) = T$ некоторая функция, то
$T$~--- статистика.
\bigskip

\textbf{Определение:}
\smallskip

Выборочными характеристиками называются функции от наблюдений (точечные оценки),
приближённо оценивающие соответствующие числовые характеристики случайной величины.
\bigskip

\subsection{Несмещённые, состоятельные и эффективные оценки}

\textbf{Определение:}

Оценка $\tilde{\varTheta}_n$ называется несмещённой оценкой параметра $\varTheta$,
если $M(\tilde{\varTheta}_n) = \varTheta$.
\bigskip

\textbf{Определение:}
\smallskip

Оценка $\tilde{\varTheta}_n$ называется состоятельной оценкой параметра $\varTheta$,
если $\tilde{\varTheta}_n$ сходится по вероятности к $\varTheta$.
\bigskip

\textbf{Определение:}
\smallskip

Оценка $\tilde{\varTheta}_n$ называется эффективной, или оптимальной, или наилучшей
несмещённой оценкой с минимальной дисперсией (НОМД), если $M(\tilde{\varTheta}_n) =
\varTheta$ и $D(\tilde{\varTheta}_n) = \displaystyle\inf_{\tilde{\varTheta}^{*}_n} D \tilde{\varTheta}^{*}_n$.
\bigskip

\subsection{Теорема о единственности эффективной оценки}

Пусть $\xi \sim L(x, \varTheta)$, $L(x, \varTheta)$~--- закон распределения известен
с точностью до параметра.

$$\overline{\varTheta} = (\varTheta_1, \dots, \varTheta_n)$$

По результатам $\mathbf{X}_1, \dots, \mathbf{X}_n$ наблюдений за $\xi$ требуется
построить оценку $\varTheta$.
\bigskip

\textbf{Теорема:}
\smallskip

Пусть $\mathbf{X}_1, \dots, \mathbf{X}_n \sim L_{\xi} (x, \varTheta)$, где
$\xi$~--- случайная величина с $M\xi = a < +\infty \;\;\; D\xi = \sigma^2$.
Тогда выборочное среднее $\overline{x} = \frac{1}{n} \sum^n_{i = 1} x_i$
является несмещённой и состоятельной оценкой $M\xi$.
\bigskip

Доказательство:
\bigskip

$$M\overline{x} = M\left(\frac{1}{n} \sum^n_{i = 1} x_i\right) = \frac{1}{n} \sum^n_{i = 1}
Mx_i =$$ 
$$= \left|x_i \text{~--- н. о. р. случайной величины} \Rightarrow
Mx_i = M\xi \;\;\; \forall i\right| = \frac{1}{n} \sum^n_{i = 1} M\xi =$$
$$= \frac{1}{n} * n * a = a \Rightarrow \overline{x}\text{~--- несмещённое}$$.

Состоятельность $\forall \epsilon > 0 \;\;\; P\{|\overline{x} - a| < \epsilon\}
\displaystyle\to_{n \to \infty} 1$
\bigskip

По неравенству Чебышёва:

$$\forall \epsilon > 0 \;\;\; P\{|\overline{x} - a| < \epsilon\} \geq 1 -
\frac{D\overline{x}}{\epsilon^2} = 1 - \frac{1}{\epsilon^2} D \left(
\frac{1}{n} \sum^n_{i = 1} x_i\right) =$$

$$=\left|\text{н. о. р.} \;\;\; Dx_i = \sigma^2\right| =$$

$$= 1 - \frac{1}{n^2}{\epsilon^2} D\xi = 1 - \frac{n \sigma^2}{n^2 \epsilon^2} =
1 - \frac{\sigma^2}{n \epsilon^2} \rightarrow_{n \to \infty} 1 - 0$$

Т.к. $P(A) \leq 1 \;\;\; \forall A$, то $\lim_{n \to \infty} P\{|\overline{x} - a| <
\epsilon\} = 1 \Rightarrow \overline{x}$~--- состоятельная.
