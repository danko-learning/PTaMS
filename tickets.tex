\subsection{Генеральная и выборочная совокупности, объём выборки.}

Рассмотрим постановку задачи математической статистики: по результатам
наблюдения за некоторой случайной величиной $\xi$ требуется сделать
выводы о неизвстном законе распределения этой величины $\mathcal{L}(x,\theta)$
либо о неизвестных парамерах $\theta_1, \dots, \theta_n$ известного распределения.
\bigskip

Пусть $\xi$~--- случайная величина с некоторой (теоретической) функцией распределения
$F_{\xi}(x) = P\{\xi < x\}, \;\;\; x \in R.$
\bigskip

\textbf{Определение:}
\smallskip

Совокупность $n$ независимых одинаково распределённых случайных величин $X_1,
X_2, \dots, X_n$ называется выборкой (выборочной совокупностью),
извлечённой из распределения случайной величины $\xi$.
\bigskip

\textbf{Определение:}
\smallskip

Под генеральной совокупностью понимается множество всех возможных значений случайной
величины $\xi$.
\bigskip

\textbf{Определение:}
\smallskip

Объёмом совокупности называется количество всех её элементов, объём выборки или
выборочной совокупности обозачается $n$, генеральной совокупности~--- $N$.
\bigskip

\subsection{Варианционный ряд, варианта, частота. Виды вариационных рядов.
Гистограмма, полигон.}

\textbf{Определение:}
\smallskip

Пусть $x_1, x_2, \dots, x_n$~--- выборка из генеральной совокупности значений.
\smallskip

Вариационным рядом называется последовательность $x_1^{*}, x_2^{*}, \dots, x_n^{*}$
элементов выборки расположенных в порядке неубывания, т.е. $x_1^{*} \leq x_2^{*}
\leq \dots \leq x_n^{*}.$ 
\smallskip

$x_i^{*}$~--- варианта.
\smallskip

$n_i$~--- частота появления варинты $x_i^{*}$ в выборке.
\bigskip

\textbf{Определение:}
\smallskip

Точечным вариационным рядом называется:

\smallskip

\smallskip

$n = \sum^m_{i = 1} n_i$, где $n_i$~--- объём выборки.
\bigskip

Для графического представления точечных вариационных рядов используется полигон

\subsection{Формулы числовых характеристик. Эмперическая функция распределения
(ЭФР). Свойства ЭФР.}

\textbf{Определение:}
\smallskip

Выборочным средним называется величина:

$$\overline{x} = \frac{1}{n} \sum^n_{i = 1} \mathbf{X_i}.$$

Если данные представлены в виде точечного или интервального вариационного ряда,
то для вычисления используют формулу:

$$\overline{x} = \frac{1}{n} \sum^n_{i = 1} x_i * n_i,$$

где $m$~--- количество групп в точечном или интервалов в интервальном вариационном
ряду, $n_i$~--- частота, т.е. количество элементов выборки, принадлежащихх $i$"=той
группе или $i$"=тому интервалу, $x_i$~--- варианта для точечного ряда и середина
$i$"=того интервала для интервального ряда.

\textbf{Определение:}
\smallskip

Выборочной дисперсией (смещённой) называется величина:

$$S^2 = \frac{1}{n} \sum^n_{i = 1} (X_i - \overline{x})^2.$$

Она характеризует среднее из квадратов отклонений наблюдаемой величины от выборочного
среднего. Величина $S = \sqrt{S^2}$ называется выборочным средним квадратическим
отклонением (смещённым) величин выборки от выборочного среднего.
\smallskip

Если данные представлены в виде точечного или интервального вариационного ряда, то для
вычисления используют формулу:

$$S^2 = \frac{1}{n} \sum^m_{i = 1} (x_i - \overline{x})^2 n_i.$$

или

$$S^2 = \left(\frac{1}{n} \sum^m_{i = 1}x^2_i n _i\right) - \overline{x}^2.$$

Здесь $m$~--- количество групп в точечном или интервалов в интервальном вариационных
рядах, $n_i$~--- частота. т.е. количество элементов выборки, принадежащих
$i$"=той группе или $i$"=тому интервалу, $x_i$~--- варианта для точечного ряда и
середина $i$"=того интервала для интервального ряда.
\bigskip

\textbf{Определение:}
\smallskip

Выборочной дисперсией (несмещённой) называется величина:

$$\overline{\sigma} = \frac{1}{n - 1} \sum^n_{i = 1} (X_i \overline{x})^2.$$

Аналогично, величина $\overline{\sigma} = \sqrt{\overline{\sigma}^2}$ называется
выборочным несмещённым средним квадратическим отклонением.
\smallskip

Очевидно, что смещённая и несмещённая выборочные дисперсии связаны формулой:

$$\overline{\sigma}^2 = \frac{n}{n - 1} S^2.$$

\subsection{Эмперическая функция распределения ЭФР. Свойства ЭФР.}

\textbf{Эмперическая функция распределения ЭФР}
\smallskip

Эмперической функцией распределения (ЭФР) называется функция

$$\tilde{F}_n(x) = \frac{1}{n} \sum^n_{i = 1} e (x - X_i),$$

где $e(x) = 1$, при $x > 0 \;\;\; e(x) = 0$, при $x \leq 0$.
\smallskip

Таким образом, если $X_i$, то $e(x) = 1$, если $X_i \geq x$,
то $e(x) = 0$, а сумма $e(x - X_i)$ будет равна количеству элементов выборки,
которые приняли значение, строго меньше некоторого $x \in R$.
\smallskip

Пусть $x_1, x_2, \dots, x_n$~--- реализация выборки $X_1, X_2,
\dots, X_n$, т.е. наблюдавшиеся значения случайной величины $\xi$.
\smallskip

Обозначим $\mu(x)$~--- число элементов выборки, строго меньших $x \in R$.
Тогда эмпирическая функция распределения $\tilde{F}_n(x)$ может быть определена
как

$$\tilde{F}_n(x) = \frac{\mu(x)}{n}.$$
\bigskip

\textbf{Свойства ЭФР:}

\begin{enumerate}
    \item $0 \leq \tilde{F}_n(x) \leq 1,$ т.к. $0 \leq \mu(x) \leq n$;
    \item неубывающая непрерывная слева функция;
    \item $\tilde{F}_n(x)$~--- ступенчатая функция для всех типов распределений;
    \item $\tilde{F}_n(x)$ сходится по распределению к $F_{\xi}(x)$.
\end{enumerate}

\subsection{Понятия статистики, оценки, выборочной характеристики.}

\textbf{Определение:}
\smallskip

Пусть $g(t_1, \dots, t_n)$~--- непрерывная функция. Оценкой $\theta$ назовём
$\tilde{\theta} = g (X_1, \dots, X_2)$.

Если $g(X_1, \dots, X_n) = T$ некоторая функция, то
$T$~--- статистика.
\bigskip

\textbf{Определение:}
\smallskip

Выборочными характеристиками называются функции от наблюдений (точечные оценки),
приближённо оценивающие соответствующие числовые характеристики случайной величины.
\bigskip

\subsection{Несмещённые, состоятельные и эффективные оценки}

\textbf{Определение:}

Оценка $\tilde{\theta}_n$ называется несмещённой оценкой параметра $\theta$,
если $M(\tilde{\theta}_n) = \theta$.
\bigskip

\textbf{Определение:}
\smallskip

Оценка $\tilde{\theta}_n$ называется состоятельной оценкой параметра $\theta$,
если $\tilde{\theta}_n$ сходится по вероятности к $\theta$.
\bigskip

\textbf{Определение:}
\smallskip

Оценка $\tilde{\theta}_n$ называется эффективной, или оптимальной, или наилучшей
несмещённой оценкой с минимальной дисперсией (НОМД), если $M(\tilde{\theta}_n) =
\theta$ и $D(\tilde{\theta}_n) = \displaystyle\inf_{\tilde{\theta}^{*}_n} D \tilde{\theta}^{*}_n$.
\bigskip

\subsection{Теорема о несмещённой состоятельной оценке мат. ожидания}

Пусть $\xi \sim L(x, \theta)$, $L(x, \theta)$~--- закон распределения известен
с точностью до параметра.

$$\overline{\theta} = (\varTheta_1, \dots, \varTheta_n)$$

По результатам $X_1, \dots, X_n$ наблюдений за $\xi$ требуется
построить оценку $\theta$.
\bigskip

\textbf{Теорема:}
\smallskip

Пусть $X_1, \dots, X_n \sim L_{\xi} (x, \theta)$, где
$\xi$~--- случайная величина с $M\xi = a < +\infty \;\;\; D\xi = \sigma^2$.
Тогда выборочное среднее $\overline{x} = \frac{1}{n} \sum^n_{i = 1} x_i$
является несмещённой и состоятельной оценкой $M\xi$.
\bigskip

Доказательство:
\bigskip

$$M\overline{x} = M\left(\frac{1}{n} \sum^n_{i = 1} x_i\right) = \frac{1}{n} \sum^n_{i = 1}
Mx_i =$$ 
$$= \left|x_i \text{~--- н. о. р. случайной величины} \Rightarrow
Mx_i = M\xi \;\;\; \forall i\right| = \frac{1}{n} \sum^n_{i = 1} M\xi =$$
$$= \frac{1}{n} * n * a = a \Rightarrow \overline{x}\text{~--- несмещённое}$$.

Состоятельность $\forall \epsilon > 0 \;\;\; P\{|\overline{x} - a| < \epsilon\}
\displaystyle\to_{n \to \infty} 1$
\bigskip

По неравенству Чебышёва:

$$\forall \epsilon > 0 \;\;\; P\{|\overline{x} - a| < \epsilon\} \geq 1 -
\frac{D\overline{x}}{\epsilon^2} = 1 - \frac{1}{\epsilon^2} D \left(
\frac{1}{n} \sum^n_{i = 1} x_i\right) =$$

$$=\left|\text{н. о. р.} \;\;\; Dx_i = \sigma^2\right| =$$

$$= 1 - \frac{1}{n^2}{\epsilon^2} D\xi = 1 - \frac{n \sigma^2}{n^2 \epsilon^2} =
1 - \frac{\sigma^2}{n \epsilon^2} \rightarrow_{n \to \infty} 1 - 0$$

Т.к. $P(A) \leq 1 \;\;\; \forall A$, то $\lim_{n \to \infty} P\{|\overline{x} - a| <
\epsilon\} = 1 \Rightarrow \overline{x}$~--- состоятельная.

\subsubsection{Определение регулярной параметрической модели. Информация Фишера.}

\textbf{Определение:}
\smallskip

Пусть $\xi$~--- наблюдаемая случайная величина.

Выборочным пространством $\mathbf{X}_n$
называется $\{\overline{X}_n = (X_1, \dots, X_n)\}$.

Параметрическим множеством называется $\varTheta$~--- множество значений параметра
$\theta$.
\bigskip

\textbf{Определение:}
\smallskip

Параметрической моделью называется

$$\left\{\mathbf{X}_n; F_{\xi}(x, \theta) : \theta \in \varTheta\right\}$$
\bigskip

\textbf{Определение:}
\smallskip

Параметрическая модель называется регулярной, если:
\begin{enumerate}
    \item параметрическое множество $\varTheta$~--- открытое множество;
    \item {носитель распределён, то есть множество $A = \{x : f(x) > 0\}$~---
    не зависит от параметров;}
    \item {$\forall \theta \;\; \text{и} \;\; x \in A$ существует производная
    $\frac{\partial f(x, \theta)}{\partial \theta} < +\infty$;}
    \item {$\forall \theta \in \varTheta \;\;\; M\left(\frac{\partial \ln f(x, \theta)}{\partial \theta}\right)
    = 0 \;\;\; M\left(\frac{\partial \ln f(x, \theta)}{\partial \theta}\right)^2 < +\infty$;}
    \item {Дважды допустимо дифференцирование под знаком интеграла по парамету.}
\end{enumerate}

\textbf{Определение:}
\smallskip

Пусть $\xi$~--- случайная величина с функцией плотности $f(x,\theta)$. Тогда
информацией Фишера для случайного вектора $\overline{X}_n$ называется
$I_n(\theta) = M(\frac{\partial}{\partial \theta} \ln f (x, \theta))^2$
\bigskip

\subsection{Неравенство Рао-Крамера. Построение эффективной по Рао-Крамеру
оценки для пуассоновского распределения}

\textbf{Неравенство Рао"=Крамера:}
\smallskip

Пусть дана регулярная параметрическая модель. Тогда для любой несмещённой оценки
$\tilde{\theta}$ параметра $\theta$ $D\tilde{\theta} \geq \frac{1}{n I_n (\theta)}$.
\bigskip

\textbf{Определение:}
\smallskip

Величину $e(\theta) = \frac{1}{n I(\theta) * D \tilde{\theta}}$ называют показаелем
эффективности по Рао"=Крамеру.

Для любой несмещённой оценки $0 < e(\theta) \leq 1$. Если $e(\theta) = 1$, то
$\tilde{\theta}_n$ называется эффективной по Рао"=Крамеру, следовательно, просто эффективна.
\bigskip

\section{Методы построения оценок}

\subsection{Метод моментов. Пример (биномиальное распределение)}

\textbf{Определение:}
\smallskip

Начальным теоретическим моментом k"=го порядка называется $m_k = M\xi^k$, если
$M|\xi|^k < +\infty$.
\bigskip

\textbf{Определение:}
\smallskip

Центральным теоретическим моментом k"=го порядка называется $\mu_k = M(\xi - M\xi)^k$,
если $M|\xi| < + \infty$.
\bigskip

\textbf{Определение:}
\smallskip

Выборочным начальным моментом k"=го порядка называется $\tilde{m}_k = \frac{1}{n}
\sum^n_{i = 1} X^k_i$.
\bigskip

\textbf{Определение:}
\smallskip

Выборочным центральным моментом k"=го порядка называется $\tilde{\mu}_k = \frac{1}{n}
\sum^n_{i = 1} (X_i - \overline{x})^k$.
\bigskip

\textbf{Метод моментов:}
\smallskip

Метод моментов состоит в том, что за оценку параметров принимается решение системы
уравнений:

\begin{equation}
    \begin{cases}
        m_k = \tilde{m}_k \\
        \mu_k = \tilde{\mu}_k
    \end{cases}
\end{equation}

\subsection{Функция правдоподобия. Метод максимального правдоподобия. Пример (биномиальное распределение)}

\textbf{Определение:}
\smallskip

Пусть $X_1, \dots, X_n \sim L_{\xi} (x, \theta)$, причём $f_{\xi} (x, \theta)$~---
известна, а $\theta$ неизвестен.

Функцией правдоподобия нахывают $L(\overline{X}_n, \theta) = \prod^n_{i = 1}
f(X_i, \theta) = f(X_1, \theta) * \dots * f(X_n, \theta)$.
\bigskip

\textbf{Метод максимального правдоподобия:}
\smallskip

Рассмотрим $\ln L(\overline{X}_n, \theta) = \sum^n_{i = 1} \ln f (X_i, \theta)$.
\smallskip

Метод максимального правдоподобия состоит в том, что за оценку параметра $\theta$
принимается точка максимума функции правдоподобия.
\smallskip

Алгоритм:
\begin{enumerate}
    \item Построить функцию $L(\overline{X}_n, \theta)$;
    \item Взять её логарифм $\ln L (\overline{X}_n, \theta)$;
    \item {$\frac{\partial \ln L (\overline{X}_n, \theta)}{\partial \theta} = 0
    \Rightarrow \theta$;}
    \item {Если значение второй производной строго меньше нля, то оценка параметра
    $\tilde{\theta}$ и есть $\theta_0$.
    
    $$\frac{\partial^2 \ln L (\overline{X}_n, \theta)}{\partial \theta^2} |_{\theta_0}
    < 0, \text{то} \;\; \tilde{\theta} = \theta_0$$}
\end{enumerate}
